############ AT Machine v.1.0 #############

Application was created on VM with ubuntu 19 
Its console application which [trying] to simulate AT Machine. (small banking system)

###### How to run it ######

Create a new folder.
Put atm.cpp file into this folder.
Open terminal
Go to this folder
And compile it with: 

*****
*
*    g++ -std=c++11 -Wall -Werror -o ./atm ./atm.cpp
*
*****

After that just use

*****
*
*    ./atm
*
*****
